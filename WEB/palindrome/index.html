Simulación: Palíndromo

Valor n -
Temp (n actual) -
Invertido (m) -
Dígito (mod) -

Ingresa un número para verificar paso a paso.

Pseudocódigo

Input: n
[x,i]=reverse_digits(n)
m=0
for j=1,...,i
     m=m+x(j)*10^(i-j)
end
if n==m
     print ‘es palindromo’
else
     print ‘no es palindromo’
end
