Simulación: Ordenamiento Descendente (Selection Sort)

Posición (i) -
Buscando (j) -
Índice Máx -
Valor Máx -

Presiona "Siguiente Paso" para comenzar el ordenamiento.

Pseudocódigo

Input: x,n
for i=1,...,n-1
    max_i=i
    for j=i+1,...,n
        if x(max_i)<x(j)
            max_i=j
        end
    end
    temp=x(i)
    x(i)=x(max_i)
    x(max_i)=temp
end
return x
