Simulación: Función a Trozos Vectorial

Definición de f(x)

$$ f(x) = \begin{cases} x^2 & \text{si } x \le -1 \\ x & \text{si } -1 < x < 1 \\ x^3 & \text{si } x \ge 1 \end{cases} $$

Vector Entrada (x)

Vector Resultado (y)

Haz clic en "Siguiente Paso" para procesar cada elemento del vector.

Pseudocódigo

Input: x,n
for i=1,...,n
    if x(i)<=-1
        y(i)=x(i)^2
    else
        if x(i)<1
            y(i)=x(i)
        Else
        y(i)=x(i)^3
        end
    end
End
Return y
