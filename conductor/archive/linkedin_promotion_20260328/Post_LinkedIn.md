🚀 ¡Comparto mi último proyecto personal!

Se trata de una plataforma interactiva diseñada para visualizar y comprender mejor los algoritmos fundamentales que forman la base de la informática y la estadística. Este proyecto fue desarrollado especialmente para los alumnos del curso de Computación Estadística I de la carrera de Ingeniería Estadística.

✅ Simulaciones en Tiempo Real: Visualiza el comportamiento de medidas estadísticas (Media, Mediana, Moda), algoritmos de ordenamiento y más.
✅ Pseudocódigo Integrado: Una vista dividida que permite seguir la lógica del algoritmo paso a paso junto a la simulación.
✅ Diseño Moderno y Responsivo: Construido con Vanilla JS, Tailwind CSS y MathJax, optimizado para cualquier dispositivo.
✅ Open Source: El código completo está disponible para que otros puedan aprender y contribuir.

Espero que esta herramienta sea útil para estudiantes, docentes y cualquier entusiasta de los algoritmos.

🌐 Demo en vivo: https://basic-algorithms.kks.qzz.io
💻 Repositorio GitHub: https://github.com/kurokirasama/basic-algorithms
