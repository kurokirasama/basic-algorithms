# Track publish_web_20260318 Specification

## Goal
Publish the existing algorithm simulation suite to the web.

## Requirements
1.  **Landing Page:** Create an `index.html` at the root that acts as a navigation dashboard.
2.  **Dashboard:** Must contain buttons/links to each of the algorithm simulation pages (`mean`, `median`, `mode`, etc.).
3.  **Title:** Page title must be "Ejemplos de Algoritmos".
4.  **Server:** Implement a `server.js` using Node.js/Express to serve the static content on port 8004.
5.  **Styling:** Maintain visual consistency with existing simulation pages (Tailwind CSS).
