# Tech Stack

## Backend
- **Node.js:** Runtime environment for the web server.
- **Express:** Minimalist web framework for serving static files.

## Frontend
- **HTML5:** Semantic structure for simulations.
- **Tailwind CSS:** Utility-first CSS framework for rapid UI development.
- **Vanilla JavaScript:** Core logic for algorithm simulations.
- **MathJax:** For rendering mathematical notation (e.g., Piecewise functions).

## Testing
- **Jest:** JavaScript testing framework.
- **Supertest:** HTTP assertion library.
- **Cheerio:** Fast, flexible, and lean implementation of core jQuery for server-side DOM testing.
